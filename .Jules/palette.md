## 2025-05-15 - [Language Consistency for Accessibility]
**Learning:** This project primarily uses Indonesian for interactive UI labels, ARIA attributes (aria-label, title), and user-facing messages (suggested prompts, greetings).
**Action:** Always implement accessibility labels and micro-interaction feedback in Indonesian to maintain consistency with the existing codebase (e.g., "Salin Pesan" instead of "Copy Message").
