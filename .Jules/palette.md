## 2025-05-15 - [Unreferenced State Variables as UX Opportunities]
**Learning:** In complex components like `ChatView.jsx`, identifying defined but unreferenced state variables (e.g., `isFooterScrolled`) can reveal missing micro-interactions that were intended by the original designers but not fully implemented.
**Action:** Always scan for unused state variables when exploring a component to find low-hanging fruit for UX improvements that align with the existing architectural intent.
