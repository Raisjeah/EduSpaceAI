## 2026-06-04 - [Scroll to Bottom Accessibility & Visibility]
**Learning:** In long-form chat interfaces, users often scroll back to reference previous context. Providing a clear, animated "Scroll to Bottom" button significantly improves navigation speed. Using Indonesian `aria-label` (e.g., "Scroll ke bawah") is consistent with the project's target demographic and existing component patterns.
**Action:** Always implement scroll-anchoring or quick-return UI in messaging components where content exceeds the viewport.
